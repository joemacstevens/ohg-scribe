Add drug names, medical acronyms, clinical terms, and brand names. Each term can be up to 6 words. Use "Paste list" to add many terms at once (one per line or comma-separated).

Features

  • Two vocabulary types: System presets (read-only, can duplicate) and User created (full edit)
  • Categories: Organize by therapeutic area or custom groupings
  • Search: Quick filter across all vocabularies
  • Term count: Shows how many words in each vocabulary
  • Tag view: Visual editing with individual remove buttons
  • Text view: Bulk editing — paste a list, one term per line
  • Paste list: Quick import from clipboard (comma or newline separated)
  • Import/Export: JSON or CSV for backup and sharing between users
  • Duplicate system presets: Copy a built-in vocabulary to customize it
  • Last edited timestamp: Track when custom vocabularies were modified
  • Access from: Settings gear icon or "Manage vocabularies" link in options panel
  • Storage: Save to app's local data directory (persists across sessions)
